# Samvad AI - Frontend

Beautiful, culturally-grounded accessibility interface for sign language translation.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Features

- Live video streaming interface with PIP signer view
- Translation deck with text input
- Avatar persona selection (Maya, Arjun, Priya)
- Adjustable signing speed
- Reverse mode toggle
- Language selection
- Real-time context analysis
- Accessibility-first design

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Material Symbols Icons

## Development

The app runs on `http://localhost:5173` by default.

All UI components are currently visual-only (Phase 2 complete). Backend integration coming in Phase 4.
